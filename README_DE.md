# 🤠 Scenario Explorer PRO (RedM / VORP)

A fully featured Admin Scenario Explorer Tool for RedM using VORP Core.

Features: - 🎥 Cinematic Preview Camera System - 🧠 Admin Group
Permission System (VORP) - 🪶 Draggable Old-West NUI Interface - ⚡
Scenario Preview / Stop / Copy Functions - 📦 Support for 3399+
Scenarios

------------------------------------------------------------------------

## 🔐 Admin System

Access only if: Character.group == "admin"

Server-side validation included.

------------------------------------------------------------------------

## 🎥 Camera System

Preview activates cinematic camera and disables it on stop.

------------------------------------------------------------------------

## 📦 Scenario Support (3399+)

Supports full RDR2 / RedM scenario pool: WORLD_PLAYER_CHORES\_*
WORLD_HUMAN\_* CAMP_SCENARIOS\_* AMBIENT\_* etc.

Note: Not all scenarios work in every build.

------------------------------------------------------------------------

## 🚀 Installation

Place resource in: resources/\[vorp\]/khali_scenario_explorer_pro

ensure in server.cfg: ensure khali_scenario_explorer_pro

------------------------------------------------------------------------

## Command

/explorer
