# SCENARIO-EXPLORER-PRO
# 🤠 Scenario Explorer PRO (RedM / VORP)

A fully featured admin scenario explorer tool for RedM using VORP Core.

## Features

-   🎥 Cinematic preview camera system
-   🧠 Admin group permission system (VORP)
-   🪶 Draggable Old-West UI
-   ⚡ Preview / Stop / Copy functions
-   📦 Supports 3399+ scenarios

------------------------------------------------------------------------

## 🔐 Admin System

Only accessible if: Character.group == "admin"

Server-side enforced.

------------------------------------------------------------------------

## 🎥 Camera System

Activates cinematic camera during scenario preview and disables on stop.

------------------------------------------------------------------------

## 📦 Scenario Support (3399+)

Supports full RDR2 / RedM scenario pool: WORLD_PLAYER_CHORES\_*
WORLD_HUMAN\_* CAMP_SCENARIOS\_* AMBIENT\_* and more.

Note: Some scenarios may not exist in all builds.

------------------------------------------------------------------------

## 🚀 Installation

Put resource into: resources/\[vorp\]/scenario_explorer

Add to server.cfg: ensure scenario_explorer

------------------------------------------------------------------------

## Command

/explorer
